Este script permite scrapear información detallada de vehículos publicados en MercadoLibre Uruguay, ingresando a cada anuncio individual.

## ¿Que extrae?
- Título
- Precio
- Ubicación real (ej. "Pocitos - Montevideo")
- Año
- Kilometraje
- Color
- Motor
- Combustible
- Transmisión
- Fecha de publicación
- Fecha del scraping
- Enlace limpio
